- 발행: 2026-06-08
- 본문:
  MAI-Code-1-Flash is a new Microsoft coding model built for fast, efficient assistance in everyday developer workflows.
  The model was trained from the ground up on clean, traceable and enterprise-grade data, without distillation from third-party models.
  With 5 billion active parameters, MAI-Code-1-Flash surpasses Claude Haiku 4.5 overall and reached 85.8% adjusted accuracy, with especially strong performance in reasoning, instruction-following, and recognizing impossible problems.

  The model is rolling out to GitHub Copilot individual users in Visual Studio Code in the model picker and under the default auto picker.
